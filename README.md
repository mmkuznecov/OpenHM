# OpenHM

This project converts video in heatmap. In framework of the hackhaton “UrbanHack” 2018.

## Getting Started

All classes are located in **"hm_class_mapping.py"**.

## Requirements

* OpenCV (Open Source Computer Vision Library)
* PIL (Python Imaging Library)
